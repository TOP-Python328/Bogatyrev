Установка программного обеспечения

  ==========  1  ==========

Для ОС Windows скачать установочный пакет MySQL Installer:
    https://dev.mysql.com/downloads/installer/

Запустить установочный пакет MySQL Installer.
Выполнить установку и конфигурацию компонентов в соответствии с рекомендациями в отправленной видеоинструкции.


Для других ОС скачать отдельные компоненты:
    https://dev.mysql.com/downloads/
    
    MySQL Community Server
    MySQL Shell
    MySQL Workbench

Установить компоненты в соответствии с требованиями вашей ОС. Инструкции по установке:
    https://dev.mysql.com/doc/refman/8.0/en/installing.html

Параметры конфигурации также описаны в видеоинструкции.


Для проверки работоспособности в клиентском приложении MySQL Shell поочерёдно выполните следующие команды (подставьте свой номер порта, если меняли):
    \connect root@localhost:3306
    \sql
    show databases;


Выполнить снимки эркана:
    1. Окно диспетчера задач с выделенной службой локального сервера MySQL
    2. Окно MySQL Shell с выполненными командами (должен быть виден вывод всех команд)
    3. Окно MySQL Workbench с выполненным входом в подключение к локальному серверу (должны быть видны панель навигации, окно редактора и т.п.)



  =========================

Снимки экрана помещаются в текущий каталог репозитория в виде JPG или PNG файлов с именами 1, 2 и 3 соответственно.
